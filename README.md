## üß™ List of Practicals (As per CD Lab Syllabus)

### **Practical 1**
**Program to implement a Lexical Analyzer in C**
- Identification of keywords, identifiers, operators, separators, and numbers
- File handling and character-based tokenization

---

### **Practical 2**
**Program to count digits, vowels, consonants, and symbols in C**
- Character classification using standard C library functions
- Demonstrates basic lexical analysis

---

### **Practical 3**
**Program to check validation of Username and Password in C**
- String handling and comparison
- Basic validation and authentication logic

---

### **Practical 4**
**Program to implement Predictive Parsing (LL(1)) in C**
- Top-down parsing approach
- Parsing table based implementation

---

### **Practical 5**
**Program to implement Recursive Descent Parsing in C**
- Grammar-based parsing
- Each non-terminal implemented as a recursive function

---

### **Practical 6**
**Program to implement Operator Precedence Parsing in C**
- Shift-reduce parsing technique
- Operator precedence relations

---

### **Practical 7**
**Program to implement LALR Parsing in C**
- Bottom-up parsing approach
- Efficient parsing technique used in real compilers

---

### **Practical 8**
**To study Lexical Analyzer Generator (LEX) and FLEX**
- Introduction to LEX/FLEX
- Structure of `.l` files
- Rule-action format

---

### **Practical 9**
**LEX Programs**
- **A.** LEX program to take input from a file and count:
  - Number of characters  
  - Number of lines  
  - Number of words  
- **B.** LEX program to count vowels and consonants in a given input string

---

### **Practical 10**
**Advanced LEX Programs**
- **A.** Print all numbers from a given file  
- **B.** Print all HTML tags from a given file  
- **C.** Add line numbers to a file and display the output
